# CodeAlpha - Exploratory Data Analysis (EDA)

## Objective

To perform Exploratory Data Analysis on a Titanic dataset and discover meaningful insights using Python.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Dataset

Titanic Passenger Dataset

## Analysis Performed

* Data Inspection
* Missing Value Analysis
* Statistical Summary
* Survival Analysis
* Age Distribution Analysis
* Gender-wise Survival Analysis

## Key Insights

1. Female passengers had a higher survival rate than male passengers.
2. Most passengers were between 20 and 50 years old.
3. Survival rates varied significantly by gender.
4. Dataset contained passenger demographic and fare information.
5. Visualizations helped identify patterns and trends.

## Conclusion

EDA was successfully performed using Python libraries and meaningful insights were extracted from the dataset.
